# Krita-OBS-One-Click-Art-Mode
Automatización para Krita y OBS. Script (.bat/.vbs) para lanzar ambos programas con un clic, iniciar grabación automática en una escena específica y pausa inteligente mediante Advanced Scene Switcher al cambiar de ventana. Optimiza el flujo de trabajo, ahorra espacio en disco y elimina la edición innecesaria. Incluye icono personalizado.
